## STLDetective用来处理什么问题
这个小脚本用来计算STL模型的包围盒，输出包围盒的长宽高参数。

## 什么是包围盒

## 如何使用这个小脚本
编译好脚本之后在命令行下输入
```bash
STLDetective.exe cat.stl
```
